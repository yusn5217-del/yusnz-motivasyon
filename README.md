# yusnz motivasyon

Modern ve şık bir motivasyon & disiplin sitesi.

**Yasal Uyarı:** Bu site eğlence, motivasyon, bilgi ve fikir amaçlıdır. Disiplin için yapılmıştır. Sitedeki puan, istatistik ve aktif kullanıcı sayıları orijinal değildir, tamamen temsilidir.

## Özellikler
- Günlük motivasyon sözleri
- Pomodoro zamanlayıcı (Odak 25 / Kısa 5 / Uzun 15)
- Günlük görev listesi
- Kaydedilen sözler
- Custom cursor, animasyonlar
- Tamamen responsive
